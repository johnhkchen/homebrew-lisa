## Install


```bash

```

## Prerequisites

Lisa requires [Zellij](https://zellij.dev/) and [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Zellij is declared as a Homebrew dependency and will be installed automatically. Claude Code must be installed separately.
